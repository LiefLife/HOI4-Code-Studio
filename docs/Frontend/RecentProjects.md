# 最近项目 (Recent Projects)

## 概述
应用启动后的主页，展示历史打开过的项目列表，支持多种布局视图。

## 功能特性
- **多种布局模式**:
  - 4列 / 3列 / 2列 网格
  - 单列列表
  - 瀑布流 (Masonry)
- **智能打开**: 点击项目卡片时，会自动检查路径有效性。如果目录存在但不是 Code Studio 项目，会提示用户进行初始化。
- **元数据展示**: 显示项目名称、路径和最后打开时间。

## 数据源
数据存储在应用配置中，通过 `getRecentProjects` API 获取。每次成功打开项目后，该列表会自动更新并将最新项目置顶。
