# 搜索功能 (Search)

## 概述
提供全局文件内容搜索能力，支持在当前项目或游戏目录中查找文本。

## 核心组件
`src/components/editor/SearchPanel.vue`

## 功能特性
- **实时防抖**: 输入停止 1 秒后自动触发搜索。
- **高级过滤**:
  - 大小写敏感 (Case Sensitive)
  - 正则表达式 (Regex Support)
  - 搜索范围切换 (Search Scope: Project / Game Directory)
- **键盘导航**: 支持上下箭头选择结果，回车跳转。
- **结果预览**: 显示匹配行号和上下文片段。

## 后端实现
- 使用 `rayon` 库进行多线程并行文件扫描，极大提高了搜索大量文本文件的速度。
- 搜索结果通过流式或分批返回给前端（具体取决于后端实现策略）。
