# 打开项目 (Open Project)

## 概述
选择并加载现有的本地文件夹作为项目。

## 核心流程
1. **目录选择**: 调用系统原生文件对话框 (`openFileDialog`).
2. **有效性检查**: 后端检查目录下是否存在 `.hoi4-code-studio` 标识文件或 `descriptor.mod`。
3. **初始化 (可选)**: 
   - 如果选择了普通的文件夹（非 Mod 项目），系统会提示用户是否将其初始化为 HOI4 Mod 项目。
   - 初始化过程会自动补充必要的元数据文件。
4. **状态加载**: 成功打开后，后端会构建文件树缓存，前端跳转至编辑器视图。
