# 设置页面 (Settings)

## 概述
应用程序全局配置界面，允许用户自定义应用行为、路径和偏好设置。

## 核心功能
- **游戏路径设置**: 配置 HOI4 游戏本体的安装路径，用于读取游戏资源和依赖。
- **更新设置**: 控制启动时是否自动检查更新。
- **启动设置**: 切换 Steam/Pirate (非正版) 启动模式。
- **界面偏好**: 
  - 最近项目列表布局 (四列/三列/两列/单列/瀑布流)
  - 配置文件存储位置 (AppData/Portable)

## 代码结构
- `src/views/Settings.vue`: 主要视图组件。
- `src/api/tauri.ts`:
  - `loadSettings()`: 加载配置。
  - `saveSettings()`: 保存配置。

## 状态管理
配置数据主要通过前端 `ref` 响应式管理，并在变更时调用后端 API 进行持久化存储。支持 "静默保存" 模式，即仅在失败时提示用户。
